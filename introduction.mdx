## Welcome to Resemblyzer App

Resemblyzer App is a voice authentication API service that leverages voice embeddings to provide secure, biometric-based user authentication. Built with FastAPI and powered by the Resemblyzer library, it enables registration and login through voice samples rather than traditional passwords.

## How It Works

<Steps>
  <Step title="User Registration">
    Users upload a voice sample (WAV, MP3, or MPEG format) along with their name. The system preprocesses the audio and generates a unique voice embedding using the Resemblyzer encoder.
  </Step>
  <Step title="Embedding Storage">
    The voice embedding is stored as binary data in PostgreSQL, associated with the user's account. This serves as their biometric signature.
  </Step>
  <Step title="Voice Login">
    During authentication, users provide their name and a new voice sample. The system generates an embedding from the new sample and compares it to the stored embedding.
  </Step>
  <Step title="Similarity Verification">
    If the cosine similarity between embeddings exceeds the threshold (0.80), access is granted. Otherwise, authentication fails.
  </Step>
</Steps>

## Technology Stack

- **FastAPI** - Modern Python web framework for building APIs
- **Resemblyzer** - Voice embedding generation and speaker verification
- **PostgreSQL** - Database for user data and voice embeddings
- **SQLAlchemy** - ORM for database interactions
- **Alembic** - Database migration management
